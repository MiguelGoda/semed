PASTA DE LOGOS - SEMED
======================

Arquivos atuais no Navbar:
---------------------------
✓ logo-semed.jpg         # Logo SEMED (aparece à ESQUERDA do navbar)
✓ logo-municipio.jpg     # Logo do Município (aparece à DIREITA do navbar)

Como funciona:
--------------
Os logos são exibidos em suas cores originais no navbar.
Se você quiser logos brancos, use versões PNG com fundo transparente
e elementos brancos, salvando-os como logo-semed.png e logo-municipio.png

Se você quiser substituir os logos:
------------------------------------
1. Substitua os arquivos logo-semed.jpg e logo-municipio.jpg
2. Pode usar formato PNG, JPG ou SVG
3. Altura recomendada: 100-150px
4. O sistema ajusta automaticamente o tamanho para 56px (h-14)

Arquivos opcionais:
-------------------
- logo-semed.png         # Versão PNG do logo SEMED
- logo-semed.svg         # Versão vetorial do logo SEMED
- logo-municipio.png     # Versão PNG do logo Município
- favicon.ico            # Favicon para o navegador

IMPORTANTE:
-----------
Para que os logos se destaquem melhor no fundo verde do navbar:
- Use logos com cores claras (branco, amarelo, etc.)
- Ou crie versões PNG com elementos brancos e fundo transparente
- O formato JPG atual mostra as cores originais dos logos
